# OpenLM


本项目致力于为大模型领域的初学者提供全面的知识体系，包括基础和高阶内容，以便开发者能迅速掌握大模型技术栈并全面了解相关知识。适用于以下对象：

1. 对大模型开发及研究感兴趣的学生；

2. 准备面试大模型相关职位的开发者。

特点：
- 系统化：从基础到高阶，全方位覆盖大模型知识体系；
- 实用性强：帮助开发者迅速上手大模型领域，提升技能；
- 针对性强：面向大模型初学者，助力顺利迈进职场。
